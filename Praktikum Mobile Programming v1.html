GENERATE APLIKASI LATIHAN FLUTTER 

 

 

1.  Silakan buat project Baru  dengan nama

Perintah cmd →  flutter create --org com.matana.tutorialflutter informatikaApp

 

Atau gunakan project yang telah ada atau telah dibuat.

2.  Letakan posisi pada folder pada informatika_tutor

Perintah cmd → cd informatika_tutor

Lihat posisi dibawah ini : 

 

3.  Lakukan inisialisasi GetX

Perintah  cmd  →  get  init:  Perintah  untuk  menginisialisasi  existing  Flutter  project,

dengan  perintah  ini  kita  bisa  mengimplementasikan  kedua               pattern               tadi               ke               project               yang

sudah ada. 

 

Pilih nomor 1.

4.  Ada  pertanyaan  selanjut  pada  folder  lib  ada  berkas  file  main.dart  akan  di  modifikasi.

Silakan pilih 1

 

5.  Maka  akan  dilakukan  generate               kembali               menggunakan               template               GetX               ,               hasilnya               seperti

dibawah ini : 

 

6.  Setelah itu silahkan buka folder tersebut dengan visual studio / android studio 

Perintah cmd → code . 

 

 

Tampilan VS Code yang terbuka.

7.  Setelah itu silahkan jalan flutter untuk akses emulator: flutter run 

Perintah cmd → flutter run

 

 

8.  Jika berhasil akan menampilkan sebagai berikut :
